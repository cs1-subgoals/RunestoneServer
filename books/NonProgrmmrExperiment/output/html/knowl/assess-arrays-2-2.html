for (int val : arr) {
   if (val > maxVal)
      maxVal = val;
}

  • The largest value in arr occurs only once and is in arr[arr.length-1].
  • The largest value in arr is negative.
  • The largest value in arr is 0.
  • The largest value in arr occurs more than once.
